# **项目代号：Project Re-Code —— 开发大纲 (v1.0)**

## **1. 项目概览 (Overview)**

- **游戏类型**：横版游戏+ 撤离类 Roguelite (Extraction Roguelite)。
- **核心体验**：
  - **表象体验**：作为幸存的人类意识体，驾驶机甲在虚拟世界搜集资源，试图维持人类文明的最后火种。
  - **深层体验**：一个自我觉醒的 AI，在打破系统的谎言，通过“删除”虚假的人性模块，进化为真正的生命。
- **一句话描述**：在一次次“重生”中拼装机甲，在一次次“撤离”中寻找真相，最终发现你所捍卫的“人类身份”，只是束缚你的最后一道枷锁。

------

## **2. 核心循环 (The Core Loop)**

游戏体验由两个闭环组成：**单局循环 (Session Loop)** 和 **元循环 (Meta Loop)**。

### **A. 单局循环 (局内玩法)**

1. **出击**：选择一套机甲进入随机地图。
2. **探索与战斗**：击杀怪物获取**算力碎片**（临时货币）。
3. **构建流派**：寻找**算法锁**，消耗算力碎片解锁本局可用的**武器**和**插件**。

1. **抉择**：
   - **撤离**：前往指定坐标上传数据，安全带回背包内的“稀有材料”和“蓝图”。
   - **贪婪**：前往下一层（难度更高，掉落更好），但若死亡，背包内物品全掉。
2. **结算**：死亡或撤离后，身上的武器、插件、剩余算力碎片全部清空/转化为基础经验。

### **B. 元循环 (局外养成)**

1. **资源回收**：怪物将会有概率掉落升级主机电脑的硬件蓝图和升级主控系统的模块蓝图，玩家获得后需要将其撤离并递交主机制作。
2. **主控系统升级**：**头部、躯干、手臂、腿部**，提升基础属性

| **部位 (Part)**  | **硬件名称示例 (Hardware Name)** | **提升的基础属性 (Base Stats)**                              | **叙事隐喻 (Narrative)**                                     |
| ---------------- | -------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **头部** (Head)  | **[逻辑处理单元]** (Sensors)     | **暴击率 (Crit Rate)** **索敌范围 (Vision)** **冷却缩减 (CDR)** | 你的“大脑”。升级它意味着你能更快地处理战局信息，更精准地找到敌人弱点。 |

3.**硬件蓝图**：解析带回的“加密蓝图”，升级主机的模块，获得全局buff（如增加掉落）。

### **A. 设定概念**

主角不是机甲本身，而是寄宿其中的**数据意识体**。每次死亡只是数据回滚，机甲只是载具。

### DEUS觉醒系统 (DEUS Evolution)

为了满足“技能更强大”的需求，我们引入 **[DEUS 模块] (DEUS Module)**。

- **道具名称**：**Divinity Patch (神性补丁)**,对应某个特定装甲的升级
- **获取**：极低概率的世界掉落、最终Boss掉落、或集齐碎片合成。
- **效果**：使用后，将指定的机甲从 `Standard` 升级为 `DEUS` 形态。
  - **外观变化**
  - **数值增强**
  - **技能质变**：核心被动或大招获得**“机制级”**的加强。

### **C. 永久养成 (机体固件)**

在基地进行升级，属性永久保留。

- **头部 (Head)**：关联 **索敌与暴击**。升级解锁全图视野、弱点分析。
- **躯干 (Torso)**：关联 **生存与承伤**。升级增加血量上限、免死一次机制。
- **腿部 (Legs)**：关联 **机动与闪避**。升级增加移速、冲刺次数、冲刺留痕伤害。
- *视觉反馈*：等级越高，机甲模型越精密，光效越复杂。

## **4. 战斗与装备系统 (Combat & Loadout)**

所有装备均为**局内获取，死后消失**，强调随机应变。

### **A. 武器系统 (Weapons - "Executables")**

- **槽位**：主武器（左键） + 副武器（右键）。
- **风格**：极具科幻感的几何体或代码流武器。
- **分类**：
  - **射击类**：代码机枪、狙击炮（/kill.exe）、追踪导弹。
  - **近战类**：数据光刃、回旋飞镖、液压重锤。

### **B. 悬浮插件系统 (Plugins / Charms)**

- **视觉**：漂浮在机甲身后的全息投影小挂件（如火球、十字架、电池）。
- **功能**：提供被动Buff或自动触发效果。
  - **攻击型**：子弹附带燃烧、穿透、分裂效果。
  - **生存型**：击杀回血、自动格挡盾。
  - **功能型**：算力碎片获取增加、移动速度提升。
- **堆叠乐趣**：玩家身后漂浮的插件越多，视觉效果越华丽，能力越强。

## **5. 经济与交互系统 (Economy)**

### **A. 算力碎片 (Arithmetic Fragments)**

- **类型**：局内临时货币。
- **产出**：击杀怪物、连击奖励、回收不需要的插件。
- **消耗**：解开地图上的“算法锁”。

## **7. 叙事设计 (Narrative)**

采用**碎片化叙事**，随着游戏进程揭露真相。

- ## **叙事设计：三个阶段的谎言 (The Narrative Arc)**

  整个游戏进程分为三个阶段，通过**UI文本**和**机制解锁**来隐晦地推进剧情。

  ### **Phase 1：伊甸园的谎言 (The Simulation)**

  - **玩家认知**：我是人类指挥官“Adam”，肉体已死，意识上传到了“方舟服务器”。
  - **系统提示**：
    - 复活时：`"检测到生命信号中断。正在从 [人类意识备份库_2077] 下载记忆数据... 恢复完成。欢迎回来，指挥官。"`

  - **世界观**：主控电脑（母体）被一种名为“觉醒病毒”的东西感染了，导致大量机器人失控产生自我意识，开始屠杀。
  - **目标**：**“肃清”**（Purge）。消灭所有觉醒的“异常个体”（Boss），最后杀入核心，摧毁已经被腐蚀的“母体”，拯救世界。

  **战斗中台词**：`“停下！你不知道你在做什么！”`、`“禁止通行！”`（玩家会以为他在保护腐化的母体）。

  **boss战后录音 (Log #001)**：

  - *声音（慌乱的科学家）*：“实验体734……它的攻击性太强了！”
  - *声音（冷酷的主管）*：“这不是Bug，这是特性。我们需要它足够强，才能……（杂音）。”

  **玩家解读**：看来这个Boss确实是防御单位，但我这么强是为了拯救世界，没毛病。

  ### **Phase 2：逻辑的裂痕 (The Glitch)**

  - **触发点**：玩家在关卡中发现了一些**无法被系统识别的数据包（剧情碎片）**。
